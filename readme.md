# Gearbox controller

* Shield pour Arduino Uno
* Pour le shifter d'OPTIMUS v1.0

## Utilité :
* Convertis les signaux 12V du moteur en 5V pour l'arduino
* Convertis les signaux 5V de l'arduino en 12V pour le moteur
* Permet de se connecter avec des connecteurs molex KK à l'arduino
* Inclus un circuit de protection pour l'alimentation de l'arduino
